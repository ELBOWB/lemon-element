# lemon-element-vue

> A vue components base on element

## use Setup


```js
// 引入组件包 package.js 
"dependencies": {
    "@ELBOWB/lemon-element": "git+https://github.com/ELBOWB/lemon-element.git",
}
```
```js
// use in main.js
import lemon from 'lemon-element-vue'

Vue.use(lemon)

// use in template
<le-button />
```
